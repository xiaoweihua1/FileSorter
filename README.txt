# FileSort 文件整理助手

## 功能
- **一键整理**：按文件类型（图片/文档/视频/音乐/压缩包）自动归类到子文件夹
- **批量重命名**：给文件批量添加前缀、日期、序号

## 使用方法

### 方法一：直接运行（推荐，不需要装Python）
1. 双击 `文件整理助手.exe` 即可打开

### 方法二：运行源代码（需要Python）
1. 安装Python（如已安装可跳过）
   - 打开 https://www.python.org/downloads/
   - 下载最新版并安装（安装时勾选 "Add Python to PATH"）
2. 按 `Win+R`，输入 `cmd`，回车
3. 输入 `pip install ttkbootstrap pillow` 回车
4. 双击 `file_sorter.py` 即可运行
