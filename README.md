## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📂 按类型分类 | 自动识别文档/图片/视频/压缩包/代码等 |
| 📅 按日期归档 | 按创建/修改日期自动归档到月份文件夹 |
| 🎨 按扩展名 | 自定义扩展名分组规则 |
| ⚡ 一键整理 | 选择目录 → 点击整理 → 完成 |
| 🔄 自动重置 | 整理完成后自动恢复默认分类名称 |

## 🚀 快速开始

```bash
git clone https://github.com/xiaoweihua1/FileSorter.git
pip install -r requirements.txt
python main.py
```
